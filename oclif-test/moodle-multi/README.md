# Usage
<!-- usage -->
```sh-session
$ npm install -g moodle-multi
$ moodle-multi COMMAND
running command...
$ moodle-multi (-v|--version|version)
moodle-multi/0.0.0 win32-x64 node-v12.18.3
$ moodle-multi --help [COMMAND]
USAGE
  $ moodle-multi COMMAND
...
```
